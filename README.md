# To-Do List App

A simple and responsive task management application built using HTML, CSS, and JavaScript.

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Track task status (Pending, In Progress, Completed)
* Set task priorities (Minor, Normal, Critical)
* Automatic progress and completion statistics
* Data persistence using Local Storage
* Dynamic greeting based on time of day

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Font Awesome

## Installation

1. Clone the repository:

```bash
git clone https://github.com/mariaali111/To-Do-List.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
